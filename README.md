# BuildingInspect

## Description

A deployable concept for a documentation tool aimed at fire inspectors. Upload a floor plan, add notes to it and export everything as a document!

## Features

- Create a user account and sign-in
- Create multiple projects and keep track of their status
- Upload one or multiple floor plans as images
- Add notes directly to the floor plan using touch or mouse
- Fully responsive interface adapted for mobile, tablet and desktop workflows
- Export all notes and floor plans as a simple template document

## Tech stack

- Frontend: TypeScript, React, Material UI
- Backend: TypeScript, Node.js, Express, MongoDB

## Known issues

As this is merely a concept there are some known issues:
- The first-time sign-in experience is a bit unintuitive, for now work around it by manually creating your first project and floor
- Backend handling of images and documents needs to be improved
- Landing page is fairly basic
